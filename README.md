# Student Management System (Full Stack)

A full-stack web application for managing student records with authentication.

## Features
- User Login & JWT Authentication
- Add, View, Update, Delete Students
- React Frontend
- REST API using Node.js & Express
- MongoDB Database

## Tech Stack
- Frontend: React, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Auth: JWT

## Run Locally
### Backend
npm install
npm start

### Frontend
npm install
npm start
